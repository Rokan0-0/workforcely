# Workforcely

Workforcely is an end-to-end modern HR Management & Self-Service Platform built with Next.js, TypeScript, Tailwind CSS, and Supabase.

## Features

- 👥 **Employee Management**: Directory, profiles, organization chart, department structures.
- ⏱️ **Attendance & Time Tracking**: Clock-in / clock-out logging, attendance history, monthly summaries.
- 🌴 **Leave Management**: Leave requests, manager approvals/rejections, entitlement tracking.
- 💵 **Payroll System**: Automatic tax (PAYE) & pension calculations, payslip generation, payout records.
- 📈 **Performance & Reviews**: Goal tracking, review cycles, performance ratings & manager feedback.
- 🎯 **Recruitment & Onboarding**: Job postings, applicant pipeline tracking, onboarding checklists.
- 📚 **Training & Courses**: Course catalog, employee enrollments, quiz evaluations & certificate generation.
- 🤖 **AI HR Assistant**: Natural language querying across company HR datasets.
- 📊 **Analytics Dashboard**: Real-time visual metrics for management.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Lucide React / Recharts
- **Database**: Supabase / PostgreSQL
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
